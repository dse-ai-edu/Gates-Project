Welcome

This AI-powered tool is specifically designed to automate the grading of questions for instructors. By supplying the system with question details and a grading rubric, it can generate scores, feedback, and confidence levels for each answer. Additionally, you can enhance its performance by providing external resources such as grading examples or related materials, which may improve grading accuracy. Please explore the two options below to experience the tool.

Create Task

The system will walk you through a step-by-step process to input the required information for creating an automatic grading task. Once all details are provided, you can begin using the tool to grade answers automatically.

Select Task

The system allows you to access your previously created automatic grading tasks. You can immediately start grading answers using the saved settings, and also view, edit, or update the task configurations as needed.
